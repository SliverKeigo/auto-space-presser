# 自动按键工具 - 可执行文件使用说明

## 执行文件版本说明

这个文件夹中的 `自动按键工具.exe` 是通过 PyInstaller 从 Python 源代码打包生成的 Windows 可执行文件。它可以在任何 Windows 系统上运行，无需安装 Python 环境。

## 使用方法

1. 直接双击 `自动按键工具.exe` 文件即可启动程序
2. 从下拉菜单中选择要自动按下的按键
3. 使用滑块调整按键间隔时间，或在精确设置框中输入具体数值
4. 点击"开始"按钮启动自动按键
5. 点击"停止"按钮停止自动按键

## 注意事项

- 首次运行时，Windows 可能会显示安全警告，这是正常的，因为该程序模拟键盘操作
- 请点击"更多信息"然后选择"仍要运行"
- 在某些游戏中使用时，可能会被游戏的反作弊系统检测到
- 程序使用时需要管理员权限以确保按键模拟正常工作

## 性能说明

- 打包的可执行文件包含了所有必要的依赖，因此文件较大，这是正常的
- 首次启动可能需要几秒钟时间

## 问题排查

如果您在使用过程中遇到问题：

1. 确保以管理员身份运行程序
2. 部分防病毒软件可能会误报，请将程序加入白名单
3. 如果在特定应用中无法正常工作，请尝试调整按键间隔

## 安全性说明

此程序完全离线工作，不收集任何用户数据，也不需要网络连接。
